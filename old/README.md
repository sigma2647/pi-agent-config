# Archived references

These directories are intentionally inactive and are not loaded by the root
`package.json`:

- `pi-interactive-subagents-upstream/` — an upstream reference checkout kept
  while its ideas are evaluated against the local implementation.
- `pi-interactive-subagents-agent-presets/` — matching upstream agent presets.

The active implementation is `extensions/subagents/`. Do not import files from
these archives into runtime code without an explicit compatibility review.
